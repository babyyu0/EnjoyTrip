'근처'는 관광지의 정보를 제공하고 지도에 노출해주는 웹사이트 입니다.<br>
사용자들은 저희 사이트를 통해 시군구별 관광지의 위치를 확인하고, 본인의 위치와 관광지의 위치가 얼마나 차이 나는지 열람할 수 있습니다.
